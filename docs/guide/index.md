# Documentation Guide

## Audience Cues

- Operators: start with the workflow and operations guides.
- Admins/IT: start with deployment, authentication, and Teams notification setup.
- Developers: start with architecture, API, database, and configuration references.
- Product and stakeholders: review the PRD for intent and scope.
